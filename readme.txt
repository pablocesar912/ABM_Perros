Proyecto Spring: ABM Perros

Proyecto "tinder de mascotas", desarrollado en el Curso de Programación
Web FullStack a través de Egg Cooperation y Gobierno de la Provincia de Mendoza.

Esta parte del proyecto desarrolla la funcionalidad de alta-baja-modificación (abm) de los mascotas
de cada usuario, perros en este caso, que formarán parte de la aplicación web.

Tecnologías aplicadas: 
- Spring Framework
- Java (maven)
- Java Script
- Html (templates/fragments)
- Css
- Thymeleaf
- JPA Repository
- Sql
